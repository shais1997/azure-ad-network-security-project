# azure-ad-network-security-project
# Enterprise Network Security & Azure AD RBAC Implementation

## 📌 Overview
This project demonstrates the design and implementation of a secure enterprise network integrated with Azure Active Directory (Entra ID). It focuses on identity-based access control, network segmentation, and layered security.

## 🚀 Key Features
- Role-Based Access Control (RBAC) using Azure AD
- Network segmentation using subnets and role-based isolation
- Multi-Factor Authentication (MFA) enforcement
- Application integration with Azure AD authentication
- Defense-in-depth security model

## 🏗️ Architecture
The network is designed using role-based segmentation across multiple floors with separate wired and wireless subnets.

## 🔐 Security Implementation
- Azure AD user and group management
- App registration and role assignment
- Conditional access and security policies
- Network-level isolation and firewall concepts

## 🛠️ Technologies Used
- Azure Active Directory (Entra ID)
- RBAC
- MFA
- Azure Policy
- Network Subnetting & Segmentation

## ✅ Learning Outcomes
- Implemented enterprise identity and access management
- Applied Zero Trust and defense-in-depth principles
- Designed secure network architecture for real-world scenarios
